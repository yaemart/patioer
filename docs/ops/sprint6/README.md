# Sprint 6 运行目录

本目录存放 Sprint 6 执行期的落地文档与每日更新记录。

## 文件清单

- `sprint6-validation-baseline.md`：环境与版本基线
- `sprint6-ac-evidence-index.md`：AC-P2-01~20 证据索引
- `daily-report.md`：每日站会日报
- `risk-rag-board.md`：风险看板（RAG）
- `command-board.md`：一页式指挥板（Top5/Top3/Top2）
- `metrics-dashboard.md`：指标看板与阈值
- `retro.md`：Sprint 6 结束复盘

## 建议更新频率

- 每天开工前：`command-board.md`、`risk-rag-board.md`
- 每天下班前：`daily-report.md`
- 每日/每 6 小时（稳定性阶段）：`metrics-dashboard.md`
- Day 10：`retro.md`
